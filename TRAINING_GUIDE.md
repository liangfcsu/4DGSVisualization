# 实时训练可视化功能使用说明

## 功能概述

现在你的4DGS播放器集成了实时训练可视化功能，可以在UI中直接启动3D Gaussian Splatting训练，并实时查看训练进度和结果。

## 使用步骤

### 1. 启动播放器

```bash
conda activate 3dgs
python player.py
```

或者加载现有场景：
```bash
python player.py data/New_Folder --sparse data/fram8/0
```

### 2. 准备训练数据

确保你的训练数据文件夹包含：
- `images/` 文件夹 - 包含训练图像
- `sparse/0/` 文件夹 - 包含COLMAP重建结果（cameras.txt, images.txt, points3D.txt）

### 3. 在UI中配置训练

1. 打开左侧控制面板
2. 展开 **"训练控制"** 部分
3. 点击 **"选择训练数据"** - 选择包含images和sparse文件夹的目录
4. 点击 **"选择输出目录"** - 选择训练结果保存位置
5. 设置 **迭代次数**（默认30000）

### 4. 开始训练

1. 点击 **"▶ 开始训练"** 按钮
2. 训练将在后台进行，不会阻塞UI
3. 实时显示：
   - **状态**: 训练当前状态
   - **进度**: 当前迭代 / 总迭代
   - **Loss**: 当前损失值
   - **点数**: 当前高斯点数量

### 5. 监控训练

训练过程中，你可以：
- 继续操作播放器（查看其他场景、调整参数等）
- 实时查看训练指标更新
- 场景视图会定期更新显示训练进度

### 6. 停止训练

如需提前停止训练：
- 点击 **"⏹ 停止训练"** 按钮
- 当前进度会被保存

## 功能特点

✅ **后台训练** - 不阻塞UI，可以同时进行其他操作
✅ **实时更新** - 每10次迭代更新一次状态
✅ **实时可视化** - 每100次迭代更新一次渲染（可调整）
✅ **自动保存** - 在指定迭代次数自动保存检查点
✅ **错误处理** - 完善的错误提示和异常处理

## 高级配置

### 修改更新频率

在 `training_manager.py` 中调整：

```python
# 状态更新频率（第203行）
if iteration % 10 == 0:  # 改为其他值，如 50
    status = {...}
    self._send_status(...)

# 可视化更新频率（在player.py中）
if data.get('iteration', 0) % 100 == 0:  # 改为其他值，如 200
    self._update_training_visualization()
```

### 修改保存迭代次数

启动训练时传递参数：

```python
save_iterations = [7000, 15000, 30000]  # 自定义保存点
```

## 输出文件

训练完成后，在输出目录中会生成：
- `point_cloud/` - 训练的点云文件
- `checkpoint_*.pth` - 训练检查点文件
- 其他训练相关文件

## 故障排除

### 训练无法启动
- 确保已安装所有依赖包
- 检查CUDA是否可用：`torch.cuda.is_available()`
- 确保训练数据路径正确

### 内存不足
- 减少迭代次数
- 降低图像分辨率
- 使用 `--resolution` 参数限制

### 训练太慢
- 确保使用GPU训练
- 减少densification频率
- 调整batch size
